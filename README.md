Tech Run - Tech-based Quiz Game

Features:
- Around 100 tech-based questions
- Audio system with sound effects and music
- Dark/Light theme support
- High score leaderboard
- Hint system (Text & 50/50 modes)
- Dynamic UI with responsive design
- Logo integration

How to play:
1. Download all the files inside the project (including the folders)
2. Make sure your Python version is up to 3.8
3. Make sure you already installed Pygame and Pillow
4. Make sure you make the folder in this format to run without issues
    TechRun/
├── main.py              # Main game script
├── config.py            # Theme and style manager
├── menus.py             # Modal windows system
├── audio.py             # Audio manager
├── data_manager.py      # High score persistence
├── questions.py         # 100 questions
├── assets/              # Logos and images
├── sounds/              # Sound effects     and music
└── build.py             # PyInstaller build script
5. Go to your ideal IDE and run main.py there
6. You're done and you can now play!
